# Sistema de Controle de Estoque - Almoxarifado de Injeção Plástica

Este é um projeto desenvolvido em JavaScript (Node.js) focado em resolver um problema real do chão de fábrica: a gestão e o controle de insumos críticos e matérias-primas na indústria de injeção plástica.

## 🚀 Funcionalidades

- **Visualização de Estoque:** Exibe o saldo atual de resinas (ex: PP Natural), pigmentos e componentes mecânicos (moldes e pinos).
- **Baixa de Materiais automatizada:** Permite que o operador retire materiais para a produção informando a quantidade exata.
- **Alertas de Estoque Crítico:** O sistema avisa em tempo real se o saldo do item atingiu o limite mínimo de segurança para evitar paradas de máquina.
- **Validação Anti-Erro:** O programa trata entradas em maiúsculo/minúsculo e previne erros comuns de digitação no terminal (como o uso de vírgulas em números decimais).

## 🛠️ Tecnologias Utilizadas

- **JavaScript / Node.js** (Ambiente de execução)
- **Módulo Readline** (Interface interativa via terminal)
